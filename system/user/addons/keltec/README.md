This custom Plugin is designed manage customizations of the Keltec Website.


# cleandescription
This function attempts to scrub the description of a product for a cleaner display

__Parameters__:
* None. Use as tag pair

__Use__:
{exp:keltec:cleandescription}{field_shortcode}{/exp:keltec:cleandescription}
